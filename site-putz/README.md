
# Site PUTZ

Site institucional do PUTZ - Barretos.

## 🚀 Como rodar o projeto localmente

1. Clone o repositório.
2. Instale as dependências:
```
npm install
```
3. Rode o servidor local:
```
npm run dev
```
4. Acesse `http://localhost:3000`

## 🌐 Deploy recomendado
- Vercel (https://vercel.com)

## ✍️ Tecnologias
- Next.js
- React
- TailwindCSS
- Framer Motion
